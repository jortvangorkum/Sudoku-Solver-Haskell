# Sudoku Solver Haskell

This sudoku solver uses depth first search.

## How to use this program

#### First build
```bash
ghc --make main.hs
```
#### Run with:
Windows:
```
./main.exe < sudokus/Easy/s01a.txt
```
Mac/Ubuntu:
```
./main < sudokus/Easy/s01a.txt
```

#### To time the execution:
```
time ./main.exe < sudokus/Easy/s01a.txt
```
